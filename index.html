Introdução

Elaborar um Trabalho de Conclusão de Curso (TCC) pode ser um grande desafio para muitos estudantes. Este guia tem como objetivo fornecer um passo a passo detalhado para a construção de um TCC sólido e bem estruturado. Aqui, você encontrará informações essenciais sobre a escolha do tema, metodologia, estrutura do trabalho e perguntas orientadoras para ajudar no desenvolvimento da pesquisa.

Sites Importantes

Alguns links interessantes para construção do tcc:

  • Google Acadêmico - Para pesquisa de artigos científicos.
  • SciELO - Base de dados de artigos acadêmicos.
  • Periódicos CAPES - Acesso a periódicos científicos.
  • Mendeley - Gerenciador de referências bibliográficas.
  • Zotero - Outra ótima ferramenta para organização de citações.
  • Canva - Para criar apresentações de slides.
  • Overleaf - Editor colaborativo LaTeX.
  • Grammarly - Para revisão de gramática em inglês.
  • Normas ABNT Fácil - Ajuda na formatação do TCC.
  • FastFormat - Formatação automática em normas acadêmicas.
  • PlagScan - Detector de plágio para garantir originalidade.
  • Scribbr - Outra opção para verificação de plágio.
